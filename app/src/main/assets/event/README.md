# event 事件类型管理

## trend.txt : 人物动向

* 格式: 每行代表一种动向
* 示例: 序号;说明;
* 0;不动, 无害;
* 1;不动, 反击;
* 2;攻击最近敌人;
* 3;优先攻击特定敌人;
* 4;破坏村庄;
* 5;宝箱;
* 6;特定抵达;
* 7;特定脱离;
* 8;破坏枯树、墙壁;
* 9;偷盗;

## tatget.txt : 过关条件

* 格式: 每行代表一种条件
* 示例: 序号;说明;
* 0;敌全灭;
* 1;制压;
* 2;特定抵达;
* 3;特定对话;
* 4;特定击破;
* 5;回合生存;
* 6;回合脱离;
* 7;回合敌全灭;
* 8;回合制压;

## event.txt : 触发事件

* 格式: 每行代表一个触发事件
* 示例: 序号;说明;
* 0;回合触发;
